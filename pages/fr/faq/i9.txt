===== Apache mod_rewrite is not enabled =====

Si vous avez le message d'erreur "//Apache Rewrite Module :    Apache mod_rewrite is not enabled//" au premier lancement dans l'interface Web de PNP4Nagios ou si vous voyez cette erreur a la place des graphiques, faites ceci:

1. Activer mod_rewrite:
<code>
a2enmod rewrite
</code>

2. Red√©marrez Apache:
<code>
/etc/init.d/apache2 restart
</code>
