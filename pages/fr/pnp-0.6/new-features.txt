===== Nouveautés de la version PNP 0.6.x =====


{{ :pnp-0.6:gallery:pnp-preview-05-08-2009.png?200|Prévu PNP 0.6.x}}

Le travail sur la version 0.6.x est toujours en cours.

Depuis la version 0.6.x nous sommes passés de subversion à GIT.  [[http://pnp4nagios.git.sourceforge.net|Le code source]] est disponible chez sourceforge.


**Fonctionnalités déjà présentes**
  * Affichage web se basant sur [[http://www.kohanaphp.com|Kohana]]
  * Affichage web se basant sur [[http://jqueryui.com/themeroller/|jQuery Themes]]
  * Fonctions javascript se basant sur des plugins [[http://jquery.com/|jQuery]]  
  * process_perfdata.pl est capable d'utiliser une base RRD par source de données
  * Amélioration de l'installateur. Possibilité de spécifier l'emplacement des layout --with-layout
  * Les erreurs RRD sont maintenant affichées en image. Plus d'image manquante.
  * Les templates PNP n'ont plus la possibilité de changer les paramètres internes
  * Les templates en version 0.4.x sont toujours utilisables
  * Fonction de génération du PDF recodée
  * Template par défaut (default.php) mieux optimisé
  * Export des bases RRD au format XML, CSV and JSON à l'aide de la fonction "xport" de RRDtool
  * Fonctions des pages recodées
  * Les pages d'erreurs vous renvoient aux pages en ligne [[http://docs.pnp4nagios.org/faq|FAQ]]
  * Popup au survol de la souris dans nagios via le plugin jQuery.clueTip
  * Support de rrdcached
