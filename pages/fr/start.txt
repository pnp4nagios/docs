====== Documentation PNP4Nagios ======

Prochainement le domain doc.pnp4nagios.org sera le point central de documentation pour PNP.

La documentation pour PNP 0.4.x a déjà été transférée de www.pnp4nagios.org.\\
La documentation pour PNP 0.6.x sera poursuivie en parallèle du développement de cette version.

====== Les contribution sont les bienvenue ======

PNP n'en serait pas là aujourd'hui sans la contribution, ces dernières années, de nombreux utilisateurs, quelques fois seulement en arrière plan.

Toute contribution est explicitement la bienvenue, même si toutes les fonctions ne peuvent pas être implémentées en une seule fois. En commençant avec la version 0.6.x qui évolue, dû au changement d'architecture.

De plus, ce domaine supporte maintenant plusieurs langues, donc je serai très content si d'autres personnes ajoutaient leur propre langue.
